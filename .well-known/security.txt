# Security Disclosure Policy

We believe in responsible security disclosure. If you discover a
vulnerability in our systems, please:

1. Email details to: hey@devradar.dev
2. Allow us 90 days to address the issue before public disclosure
3. Provide sufficient information for us to reproduce the issue
4. Do not exploit the vulnerability for any reason
